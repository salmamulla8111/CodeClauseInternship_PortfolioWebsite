Project 1-Food Ordering System

Project 2- Word to PDF Converter

A Word to PDF converter developed using Android Studio, Java, and XML is a mobile application designed to convert Microsoft Word documents into PDF files directly on an Android device. Utilizing the Android Studio development environment, Java serves as the primary programming language, while XML handles the user interface design. Users can select a Word document from their device, initiate the conversion process, and obtain a PDF version of the file. The application employs various functionalities such as file selection, conversion algorithms, and PDF generation capabilities. The development process involves creating a user-friendly interface through XML layouts, implementing file handling and conversion logic using Java, and ensuring compatibility and performance optimization for Android devices.
